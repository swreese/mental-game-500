# RankSongs Project

## Overview
The RankSongs project is a web application dedicated to showcasing the music and story of the Texas pop artist, Livingston (Drake Jon Livingston Jr.). The application allows users to explore his biography, listen to his music, and participate in ranking his songs.

## Project Structure
```
rank-songs-api
├── index.html          # Main entry point for the application
├── liv_home.html       # Artist biography and music showcase
├── mentalgame500.html  # New page for additional content
├── liv_ranks.html      # Ranking system for Livingston's songs
├── liv_global.html     # Global rankings and statistics
├── FAQs.html           # Frequently asked questions about Livingston
├── css
│   └── styles.css      # Styles for the project
├── js
│   └── main.js         # JavaScript functionality for interactivity
├── data
│   └── songs.json      # JSON data for songs
├── partials
│   ├── header.html     # Reusable header section
│   └── footer.html     # Reusable footer section
├── .gitignore          # Files to be ignored by version control
├── package.json        # npm configuration file
└── README.md           # Project documentation
```

## Features
- **Artist Showcase**: Detailed information about Livingston, including his journey and music.
- **Song Rankings**: Users can view and submit rankings for Livingston's songs.
- **Global Statistics**: Display of global rankings and statistics related to Livingston's music.
- **FAQs**: Answers to common questions about Livingston and his music.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser to view the application.

## Usage
- Explore the different pages to learn more about Livingston and his music.
- Participate in ranking songs and view how they compare globally.
- Check the FAQs for any questions regarding the artist and his work.
